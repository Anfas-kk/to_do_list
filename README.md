To-Do List Web App

This is a simple and interactive To-Do List application built using HTML, CSS, and JavaScript. It allows users to add, manage, and remove daily tasks with a clean and modern UI.

Check the preview:  https://anfas-kk.github.io/to_do_list/

🚀 Features
Add new tasks easily
Mark tasks as completed using checkboxes
Completed tasks are visually updated (strikethrough + color change)
Remove tasks with a delete (×) button
Simple and responsive design with gradient background
Interactive button effects

🛠️ Technologies Used
HTML – Structure of the app
CSS – Styling and layout design
JavaScript – Dynamic task management

⚙️ How It Works
User enters a task in the input field
Clicking the Submit button adds the task to the list
Each task includes:
A checkbox to mark completion
A delete button to remove the task
JavaScript dynamically creates and updates DOM elements

📂 Project Structure
Single HTML file containing:
Inline CSS for styling
Inline JavaScript for functionality

⚠️ Note
Tasks are not stored permanently (no local storage or database used)
Refreshing the page will clear all tasks
